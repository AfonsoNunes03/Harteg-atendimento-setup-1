Harteg Atendimento IA

Antes de começar o setup prático, aqui está um resumo do que vais construir nesta sessão.

O que vais construir
📱
WhatsApp ligado — o teu número a responder a leads via Evolution API (grátis) ou Z-API
📧
Email automatizado — integração Resend com plano gratuito de 3.000 emails/mês
🤖
Agente IA BANT — qualifica leads automaticamente (OpenAI, Gemini ou Claude)
👥
Base de contactos — importa a tua lista e organiza-a numa base local
🚀

Etapa 1 de 10 Pendente

Instalar Pré-requisitos

Antes de começar, precisas de ter alguns programas instalados. Não te preocupes — o Claude verifica tudo automaticamente e avisa-te se estiver a faltar algo.

O que precisas de ter instalado
1
Claude Code CLI — o assistente que vai conduzir o teu setup
2
Python 3.10+ — linguagem dos scripts de automação
3
Node.js 18+ — necessário para algumas dependências
4
Git — para clonares o repositório
5
Docker Desktop — para correr a Evolution API (WhatsApp)
Como instalar cada programa
1

Claude Code — o assistente de setup

Abre o terminal e cola o comando abaixo:

terminal
npm install -g @anthropic-ai/claude-code
💡 Não tens conta Anthropic? Cria uma gratuitamente em claude.ai antes de continuar.
2

Python, Node.js, Git e Docker Desktop

Escolhe o teu sistema operativo e segue as instruções:

🍎
macOS — usa o Homebrew: brew install python@3.11 node git && brew install --cask docker. Depois abre o Docker Desktop e aguarda que inicie.
🪟
Windows — descarrega e instala: Python (python.org), Node.js (nodejs.org), Git (git-scm.com) e Docker Desktop (docker.com). Reinicia o computador depois de instalar o Docker.

Marca como concluída quando todos os programas estiverem instalados.

Etapa 2 de 10 Pendente

Primeiro Acesso — Clonar e Abrir

Com os pré-requisitos instalados, vais clonar o repositório e abrir o Claude. A partir daí, o Claude conduz tudo sozinho — não precisas de digitar mais nenhum comando.

O único comando que precisas de correr
terminal — cola e prime Enter
git clone <URL_DO_TEU_REPOSITORIO_HARTEG> && \
cd harteg-atendimento-setup-1 && \
claude
1

O que acontece quando corres esse comando

O Git descarrega o repositório, entra na pasta e abre o Claude Code. O Claude vai apresentar-se e aguardar que dês o sinal para começar.

2

Na primeira vez, o Claude vai autenticar-se

Se for a primeira vez que usas o Claude Code, ele vai pedir-te para fazeres login com a tua conta Anthropic. Segue as instruções no ecrã (é só abrir um link no browser).

💡 Não tens conta Anthropic? Cria uma gratuitamente em claude.ai antes de continuar.
3

Escreve INICIAR SETUP para começar

Quando o Claude aparecer e se apresentar, escreves exatamente isto e primes Enter:

escreve no Claude
INICIAR SETUP

A partir daí o Claude conduz tudo — não precisas de digitar mais nenhum comando técnico.

Marca como concluída quando escreveres INICIAR SETUP e o Claude começar o setup.

Etapa 3 de 10 Pendente

Configurar o Ambiente

O Claude vai recolher algumas informações sobre ti e o teu negócio, e criar toda a estrutura de pastas da tua operação IA no computador.

O que o Claude vai perguntar
1

O teu nome

Para personalizar o agente e os relatórios com o teu nome.

2

Nome do teu negócio

Usado nas mensagens do agente e nos relatórios de status.

3

O Claude vai perguntar qual o provedor de WhatsApp que queres usar

Tens duas opções — escolhe a que melhor se encaixa no teu cenário:

🚀
Evolution API
Corre no teu computador via Docker. Open-source e sem custo adicional. Recomendado para quem quer começar sem gastar.
Gratuita
📱
Z-API
Corre na nuvem — mais estável e não depende do teu computador ficar ligado. Serviço brasileiro — confirma disponibilidade e preço para Portugal.
Fallback opcional
O que é criado na tua máquina
estrutura criada em ~/.operacao-ia/
~/.operacao-ia/
├── config/
│   └── config.json          # as tuas configurações
├── scripts/               # scripts de automação
├── data/                  # base de contactos
└── logs/                  # relatórios e logs

Marca como concluída quando o Claude confirmar que a pasta foi criada.

Etapa 4 de 10 Pendente

Ligar o WhatsApp

O Claude vai instalar e configurar o provedor de WhatsApp escolhido. Vais escanear um QR Code com o teu telemóvel para ligar o número.

Como funciona a ligação
1

Evolution API — corre no teu computador

O Claude instala e configura tudo automaticamente. Vai mostrar um QR Code para escaneares com o WhatsApp do telemóvel.

⚠️ O Docker Desktop precisa de estar aberto antes desta etapa. Se ainda não estiver, abre-o agora e aguarda que o ícone estabilize na barra de tarefas.
2

Z-API — corre na nuvem (fallback opcional)

O Claude vai pedir o teu Instance ID e Token. Encontras esses dados no painel em app.z-api.io depois de criares a tua conta.

💡 Vantagem da Z-API: funciona 24h mesmo com o teu computador desligado.
3

Escaneia o QR Code (apenas Evolution API)

O QR Code aparece diretamente na janela do Claude. Abre o WhatsApp no telemóvel → Dispositivos Ligados → Ligar Dispositivo → escaneia o código.

Marca como concluída quando o Claude confirmar que o WhatsApp está ligado.

Etapa 5 de 10 Pendente

Configurar o Email

Vamos ligar o email para envio de notificações e comunicações. Usamos a Resend, uma plataforma fiável com plano gratuito generoso.

Como obter a tua API Key da Resend
1

Cria uma conta gratuita em resend.com

O plano gratuito inclui 3.000 emails/mês e 100/dia — mais do que suficiente para começar.

2

Acede a API Keys no painel da Resend

No menu lateral: API Keys → Add API Key. Dá um nome como "Harteg" e copia a chave gerada.

3

Cola a API Key quando o Claude pedir

O Claude vai pedir a chave e enviar um email de teste para confirmar que está tudo a funcionar.

⚠️ Guarda a API Key num local seguro — ela só aparece uma vez na Resend.

Marca como concluída quando o email de teste chegar à tua caixa de entrada.

Etapa 6 de 10 Pendente

Ativar o Agente IA

Aqui configuras o agente IA que vai responder aos teus leads no WhatsApp usando a metodologia BANT — qualificando e encaminhando para o fecho automaticamente.

Escolhe o teu provedor de IA
⚡
OpenAI
gpt-4o-mini. Rápido e barato para respostas no WhatsApp. Sem tier gratuito.
Pago por uso
✨
Google Gemini
Gemini Flash. Tier gratuito disponível fora da UE/Reino Unido/Suíça — confirma a tua região antes de escolher.
Grátis (fora da UE)
🧠
Anthropic (Claude)
Claude Haiku. O mais natural nas respostas. Pago por uso, sem restrição geográfica.
Melhor qualidade
O que o Claude vai configurar
1

API Key do provedor escolhido

O Claude vai pedir a tua chave e testá-la com uma mensagem simples para garantir que está a funcionar.

2

Nome e tom do agente

Como o agente se vai apresentar aos teus leads: nome, se é mais formal ou informal, foco em vendas ou suporte.

3

O agente fica instalado em ~/.operacao-ia/scripts/

Com um script de arranque pronto a correr quando quiseres ativar.

Marca como concluída quando o Claude confirmar que a API Key funcionou.

Etapa 7 de 10 Pendente

Importar os teus Contactos

Importa a lista de contactos a quem queres enviar mensagens. O sistema aceita CSV ou colagem direta — sem precisares de uma folha de cálculo online.

Formas de importar
1

Colar diretamente no terminal

O Claude vai pedir que coles os teus contactos no formato abaixo, um por linha. Quando terminares, deixa uma linha em branco e prime Enter.

formato de colagem
# nome,telefone (com indicativo do país, sem espaços)
João Silva,351912345678
Maria Santos,351923456789
Carlos Oliveira,351934567890
2

Indicar o caminho de um ficheiro CSV

Se tiveres um ficheiro CSV, indica o caminho completo (ex: ~/Downloads/contactos.csv). O Claude deteta o separador automaticamente.

3

O que acontece automaticamente

Os telefones são validados e normalizados (indicativo +351 adicionado automaticamente). A lista fica guardada em ~/.operacao-ia/data/contacts.db.

Marca como concluída quando o Claude mostrar o resumo dos contactos importados.

Etapa 8 de 10 Pendente

Configurar Disparos

O sistema de disparo envia mensagens para a tua lista com proteção automática contra bloqueio. Revisas sempre antes de enviar qualquer coisa.

Proteção automática contra bloqueio
🛡️
Máximo 30 mensagens por hora — ritmo seguro para evitar banimento
🛡️
Máximo 150 mensagens por dia — limite diário para uma operação saudável
🛡️
90 segundos entre mensagens — intervalo mínimo entre envios
🛡️
Deduplicação automática — nunca envia a mesma mensagem duas vezes ao mesmo contacto
Como usar o dispatcher
1

Pede ao Claude para montar o disparo

Dentro do Claude, dizes a mensagem que queres enviar e para quem. O Claude monta tudo e mostra uma pré-visualização com os primeiros contactos antes de enviar qualquer coisa.

💬 Exemplo do que dizer ao Claude:
"Claude, quero enviar para a minha lista: Olá {nome}, tudo bem? Tenho uma novidade para te contar. Mostra o preview antes de enviar."
2

Revê o preview e confirma

O Claude mostra como vai ficar cada mensagem com o nome real de cada contacto. Só depois de confirmares é que ele começa a enviar — nunca sem a tua aprovação.

⚠️ O sistema tem proteção automática: máx. 30 mensagens/hora e 150/dia. Isto evita o bloqueio do teu número.
3

Filtra por grupo ou tag quando quiseres

Podes pedir ao Claude para enviar só para um grupo específico de contactos. Exemplo: "Envia só para os contactos com a tag 'clientes ativos'."

Marca como concluída quando o Claude confirmar que o dispatcher foi instalado.

Etapa 9 de 10 Pendente

Ativar o Monitor Diário

O monitor verifica automaticamente se todos os serviços estão a funcionar e envia-te um relatório no próprio WhatsApp todos os dias às 8h da manhã.

O que o monitor verifica
📡
WhatsApp ligado — verifica se a instância está ativa
📧
Email a funcionar — teste silencioso de envio
🤖
Agente IA a correr — verifica se o processo está ativo
📊
Relatório HTML — guardado em ~/.operacao-ia/logs/ com histórico
Instalação automática
1

O Claude instala o monitor automaticamente

O monitor fica registado como um serviço do sistema e começa a correr todos os dias às 8h — sem precisares de fazer nada.

2

Primeiro health check imediato

O Claude vai perguntar se queres executar o monitor agora para veres o estado de tudo o que foi configurado.

Marca como concluída quando o monitor estiver instalado e o primeiro health check mostrar tudo verde.

Etapa 10 de 10 Pendente

Teste Final e Atendimento IA Pronto

Último passo! O Claude vai guiar-te pelo primeiro teste de disparo e confirmar que o Harteg Atendimento IA está completamente funcional.

O que acontece nesta etapa
1

Resumo de tudo o que foi configurado

O Claude mostra um painel completo com o estado de cada componente: WhatsApp, Email, Agente IA, Contactos, Dispatcher e Monitor.

2

Primeiro disparo guiado — dentro do Claude

O Claude conduz-te pelo primeiro teste de envio. Mostra como vai ficar a mensagem para cada contacto antes de qualquer coisa ser enviada. Aprovas e ele dispara.

💬 Basta dizeres ao Claude:
"Faz o teste do primeiro disparo para a minha lista."
Ele trata do resto.
3

Harteg Atendimento IA pronto

A partir de agora, a tua operação funciona de forma autónoma. O agente responde a leads, o monitor avisa se algo correr mal, e disparas quando quiseres.
